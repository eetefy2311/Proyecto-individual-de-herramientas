# Proyecto-individual-de-herramientas
Este repositorio es para subir el proyecto de R que se trabajó en RStudio, respecto al blanqueamiento de arrecifes de coral.

En la carpeta info se agregaron los archivos del reporte y análisis de gráficos correspondientes, en la carpeta code se encuentra todo el código que se utilizó en el proyecto y en la carpeta res se encuentran las visulaizaciones de los gráficos implementados para el proyecto.
